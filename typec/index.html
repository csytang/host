Description

TypeC is varaibility-aware IDE for configurable systems/product lines. The TypeC system is build upon Eclipse IDE by extending the basic framework of Eclipse CDT. TypeC allows developers to create the program database for the system proccessed along with extracting all valid configurations. Experimental Results

We have released the experimental results of all subject systems at: our google drive. In the repos, each folder contains two files: (1) $project and (2)$project_analysisresult. In $project file, it contains (1) the url for downloading source code of the project; (2) how to compile the project; and (3) steps to build the project in Eclipse. In $project-analysisresult, it contains all basic information of this project by metrics, and configuration profiles.
