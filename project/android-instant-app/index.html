Description

This project hosts for several research output for Android Instant App.

Tools

This project contains several subtools and components as follows:

  • GooglePlaySpider: is a spider tool (in python) for automatically extract App information from Google Play. Specifically, we instant a script to filter the Instant Apps.
  • IADnP: (Instant App Downloader aNd Parser) is a spider and parser tool (in python) for automatically download and parse instant apps from Google Play.
  • asPL: the Android Studio Plugin for instant app research.
  • FroidSpider: this spider will download all urls for open source Android application.
* Note, if you need the source code for your research work or industry project, please contact the developer at chrisyttang(AT)hotmail.com.

Copyright and Research Ethics

  • Evaluate the legal situation of downloading and working on copyrighted applications with regards to their situation (local laws, host institutions policy, etc);
  • Do not, in general, redistribute this data without our consent;
  • Do not, in particular, use this data to create an application market;
  • Do not make a commercial usage of this data;
  • Get a faculty, or someone in a permanent position, to agree and commit to those conditions;
