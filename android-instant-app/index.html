Description

This project aims at finding the vulnerability of Google Instant App scheme, which is promoted at Google I/O 2016 conference.

Download

This project contains several subtools and components as follows:

  • GooglePlaySpider: is a spider tool (in python) for automatically extract App information from Google Play. And specifically, we instant a script to filter the Instant Apps.
